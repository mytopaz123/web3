Javascript

JavaScript (/ ˈdʒɑːvəˌskrɪpt /), [6]는 종종 JS로 축약되며 ECMAScript 사양을 준수하는 프로그래밍 언어입니다. [7] JavaScript는 종종 적시에 컴파일되며 다중 패러다임입니다. 중괄호 구문, 동적 타이핑, 프로토 타입 기반 객체 지향 및 일류 함수가 있습니다. HTML 및 CSS와 함께 JavaScript는 월드 와이드 웹의 핵심 기술 중 하나입니다. [8] JavaScript는 대화식 웹 페이지를 가능하게하며 웹 응용 프로그램의 필수 부분입니다. 대부분의 웹 사이트는 클라이언트 측 페이지 동작에이 웹 사이트를 사용하며 [9] 모든 주요 웹 브라우저에는 전용 JavaScript 엔진이 있습니다.
