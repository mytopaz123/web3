HTML

the standard markup language for creating web pages and web applications. HTML 요소는 HTML 페이지의 빌딩 블록입니다. HTML 구성을 사용하면 이미지 및 대화 형 양식과 같은 기타 개체가 페이지에 포함될 수 있습니다. HTML은 제목, 단락, 목록, 링크, 따옴표 및 기타 항목과 같은 텍스트의 구조적 의미를 표시하여 구조화 된 문서를 작성하는 수단을 제공합니다.

HTML 요소는 꺾쇠 괄호를 사용하여 태그로 표시됩니다. 콘텐츠를 페이지에 직접 소개하는 태그 서라운드와 같은 다른 태그는 문서 텍스트에 대한 정보를 제공하며 다른 태그를 하위 요소로 포함 할 수 있습니다. 브라우저는 HTML 태그를 표시하지 않지만이를 사용하여 페이지 내용을 해석합니다. HTML은 JavaScript와 같은 스크립팅 언어로 작성된 프로그램을 포함 할 수 있으며 이는 웹 페이지의 동작 및 내용에 영향을줍니다. CSS를 포함하면 내용의 모양과 레이아웃이 정의됩니다.
